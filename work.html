06 - Coding | Frank Slide Sector Map

×

This project is near and dear to my heart. I am an advide climber and was tired of
having no maps of my favorite outdoor climbing spot, Frank Slide.
In turn I devised this as a way to map out the frank slide climbing sectors.
This project a day to complete I had to build the map, turn it into XML then
devise and API from that to allow myself to create a web based app that has had
now over 1000 views. The data from my repository where the web app is hosted
is to the left and uses AJAX to get real time updates on things such as Stars, Forks, and Open Issues.
The information adjacent to it is a custom made API to pull the cordinates from my map, I did this by adding a
data.json file to my frank slide map repository and then used AJAX to pull the data from it. All and all I am incredibly proud of this project and as time goes on I will continue to go forward and create this project into something even bigger and better.
